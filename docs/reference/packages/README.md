# Package References

This area holds short local reference notes for external packages that matter to repeated operational work in `NEXUS-EventStore`.

This repo inherits the broader package-guidance rule from `NEXUS-EMERGING`:

- official package docs first
- local package notes second
- local overrides only when this repo has stronger package-specific operational workflows or gotchas

Right now this repo does not need a seeded package note, but the folder exists so future recurring operational dependencies can be documented without inventing the pattern again.
